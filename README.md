# Mindo - Obsidian 无限画布思维导图插件

Mindo 是一个基于 React 的 Obsidian 插件，提供了一个无限画布，支持创建节点、拖拽连线、AI 创意扩展、图片粘贴等功能，帮助你更好地整理思绪。

## ✨ 主要功能

*   **无限画布**: 自由拖拽平移、缩放，无拘无束地挥洒创意。
*   **节点编辑**: 双击创建节点，支持标题和详细内容（支持 Markdown 渲染）。
*   **灵活连线**: 支持贝塞尔曲线、直线、折线，可自定义箭头方向、线条样式和颜色。
*   **AI 赋能**: 集成 Google Gemini / OpenAI / DeepSeek，一键扩展思维，生成创意子主题。
*   **分组管理**: 框选多个节点进行编组，整体移动和管理。
*   **图片支持**: 直接粘贴剪贴板中的图片到画布。
*   **导出图片**: 支持高清导出为 PNG 图片。
*   **暗色模式**: 完美适配 Obsidian 的深色主题。

## 🚀 快速开始

### 1. 配置 AI (推荐)
为了使用强大的 AI 扩展功能，请先进行配置：
1.  进入 Obsidian **设置 (Settings)** > **Mindo**。
2.  **AI 服务商预设**: 选择您使用的服务商 (Google Gemini, DeepSeek, OpenAI 或 自定义)。
3.  **API 密钥**: 输入您的 API Key。
4.  **模型名称**: (可选) 根据需要修改模型名称，如 `gemini-2.0-flash` 或 `gpt-4`。

### 2. 创建导图
*   **方法一**: 点击 Obsidian 左侧边栏的 "脑回路" 图标 (新建思维导图)。
*   **方法二**: 使用命令面板 `Ctrl/Cmd + P`，搜索并执行 `Mindo: 新建思维导图`。
*   输入文件名后即可创建。

### 3. 基本操作

#### 画布控制
*   **平移**: 按住 **鼠标中键** 拖拽，或按住 `Ctrl/Cmd` + **鼠标左键** 拖拽，或者按住 `Space` 空格键拖拽。
*   **缩放**: 按住 `Ctrl/Cmd` + **鼠标滚轮**，或使用下方工具栏的 `+` / `-` 按钮。
*   **重置**: 点击下方工具栏的 "重置" 按钮回到中心，或 "适应视图" 查看所有节点。

#### 节点操作
*   **创建节点**: 在画布空白处 **双击** 鼠标左键。
*   **编辑文本**: 
    *   单击选中节点。
    *   再次点击标题或内容区域进入编辑模式。
    *   按 `Ctrl + Enter` 或点击空白处完成编辑。
*   **调整尺寸**: 拖拽节点右下角的缩放手柄（需节点有内容或为图片时可见）。
*   **更改颜色**: 选中节点后，点击上方悬浮工具栏中的颜色圆点。
*   **删除节点**: 选中节点后按 `Delete` / `Backspace` 键，或点击悬浮工具栏的垃圾桶图标。

#### 连线操作
*   **创建连线**: 鼠标悬停在节点上，边缘会出现 4 个圆点（连接手柄）。按住手柄拖拽至另一个节点即可连线。
*   **调整连线**:
    *   **选中连线**: 点击线条，会弹出属性面板。
    *   **修改样式**: 支持切换 实线/虚线/点线，直线/曲线/折线，以及箭头方向。
    *   **添加标签**: 在属性面板输入框中输入文字。
    *   **调整路径**: 
        *   双击连线可添加控制点（Breakpoints）。
        *   拖拽控制点可改变线条走向。
        *   双击控制点可将其删除。
    *   **重连**: 拖拽连线的端点到另一个节点的连接点上。

### 4. 高级功能

#### ✨ AI 创意扩展
当您思绪卡壳时，让 AI 帮您发散思维：
1.  选中一个节点（作为中心主题）。
2.  点击底部工具栏紫色的 **"AI 扩展"** 按钮。
3.  稍等片刻，AI 将自动生成 4 个相关的子节点并连接。

#### 🖼️ 图片支持
*   **粘贴图片**: 复制任意图片（网页右键复制、系统截图等），在画布上按 `Ctrl/Cmd + V` 即可粘贴。
*   图片节点会自动调整大小以适应图片比例。

#### 📦 编组 (Group)
1.  按住鼠标左键在画布上拖拽出一个选框，选中多个节点。
2.  点击底部工具栏的 **"编组"** 图标。
3.  系统会自动创建一个包含这些节点的虚线框分组。
4.  拖拽分组标题可整体移动内部节点。

#### 📐 对齐
1.  框选多个节点。
2.  点击底部工具栏出现的 **水平对齐** 或 **垂直对齐** 按钮，让排版更整齐。

#### 📷 导出
*   点击底部工具栏的 **图片** 图标。
*   选择导出倍率 (1x - 4x)，倍率越高清晰度越高。

## ⌨️ 快捷键速查

| 快捷键 | 功能 |
| :--- | :--- |
| `双击画布` | 新建节点 |
| `Ctrl / Cmd + 滚轮` | 缩放画布 |
| `Ctrl / Cmd + 拖拽` | 平移画布 |
| `Delete / Backspace` | 删除选中项 |
| `Ctrl / Cmd + V` | 粘贴图片 |
| `Ctrl / Cmd + Enter` | 完成节点编辑 |
| `Esc` | 取消选中 / 退出编辑 |
